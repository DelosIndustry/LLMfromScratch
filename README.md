# LLM From Scratch Skeleton

This folder is organized in a chapter-style layout inspired by
`gilbutITbook/080421`.

The structure mirrors the reference repository at a high level:

- `setup/`: environment and installation notes
- `ch02/` to `ch07/`: main chapter workspaces and bonus material
- `appendix-A/`, `appendix-D/`, `appendix-E/`: appendix workspaces

The directories are intentionally empty skeletons. Add notebooks, scripts,
datasets, and notes to the matching chapter folders as the project grows.
